# DTEK_snake — «Змійка» у стилі графіка відключень ДТЕК

Мінімальна веб‑гра, що імітує вигляд таблиці відключень ДТЕК: 7 днів × 24 години, вертикальні підписи годин і поділ кожні 3 години. Змійка рухається по «клітинках розкладу», їжа позначена жовтим, інтерфейс поки тільки українською.

## Живе демо
- Переглянути на GitHub Pages: [DTEK_snake](https://mstohnii.github.io/DTEK_snake/)

## Запуск
- Відкрийте файл `index.html` у браузері (Chrome, Edge, Firefox, Safari).
- Ніяких залежностей або збірки не потрібно.

## Можливості
- 7×24 сітка з українськими підписами днів та «Часові проміжки».
- Вертикальні інтервали годин виду `00–01, 01–02, …`; товстіші роздільники кожні 3 години.
- Іконки з файлів: `snake.svg` (змійка) та `target.svg` (ціль/їжа).
- Адаптивність: квадратні клітинки, вирівнювання підписів.
- Керування з клавіатури і на мобільних (свайпи).
- Пауза, вікно завершення гри з рахунком.
- Рекорд (найкращий результат) зберігається у `localStorage`.
- Динамічна складність: гра починає повільно і поступово пришвидшується з кожним з’їденим елементом.

## Керування
- Клавіатура: стрілки або WASD.
- Підтримка інших розкладок (UA/RU): Ц/Ф/І(Ы)/В.
- Пробіл — пауза/продовження.

## Налаштування
- Виділений день тижня — у файлі `script.js`, змінна `HIGHLIGHT_ROW` (0 = Понеділок … 6 = Неділя).
- Швидкість: `BASE_TICK_MS`, `MIN_TICK_MS` та формула у `getTickMs()`.
- Стилі (кольори, межі, відступи) — у `styles.css`.

## Технічні деталі
- Чисті HTML/CSS/Vanilla JS, без зовнішніх бібліотек.
- Сітка — CSS Grid; логіка гри — у `script.js`.
